# System Optimizer для Windows 10/11

Красивое приложение для мониторинга ресурсов и глубокой оптимизации системы.

## Возможности
- Отображение загрузки CPU, GPU, температуры, частоты
- Информация о RAM (объём, скорость)
- Мониторинг дисков (заполненность, скорость чтения/записи)
- **Оптимизация:**
  - Очистка временных файлов, кэша миниатюр, шрифтов
  - Освобождение рабочего набора процессов
  - Сброс сетевых настроек, TCP/IP оптимизация
  - Установка схемы высокой производительности
  - TRIM для SSD, повышение приоритетов процессов
  - Остановка тяжёлых служб, завершение фоновых приложений
  - Проверка обновлений драйверов (требуются права администратора)

## Установка
Скачайте последний релиз (`SystemOptimizer.exe`) и запустите от имени администратора.
## Скачать [ [здесь](https://github.com/stradanchenkov/system-optimizer/releases/tag/v1.2) ]

## Сборка из исходников
```bash
pip install customtkinter psutil wmi nvidia-ml-py send2trash pyinstaller
pyinstaller --onefile --windowed --uac-admin --collect-data customtkinter init.py
```
